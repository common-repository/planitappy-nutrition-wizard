== Description ==

Get your client's dietary requirements for a nutrition plan direct from your WordPress website using the PlanItAppy Nutrition Wizard plugin.

= What this plugin does =

This plugin displays a Nutrition Plan Wizard Form on your website using a provided shortcode. You input your PlanItAppy store URL, and it displays the questionnaire using an iFrame. This involved calling Javascript from an external server. Any end user who wishes to make submit their answers will have to either register an account, or enter a username and password for an existing account on your PlanItAppy web store.

== Installation ==

1. Upload the plugin ZIP file to the `/wp-content/plugins/` directory
2. Unzip the plugin ZIP file.
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Read the instructions under the plugins settings.
